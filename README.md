# VaryAble Portfolio

Production-grade SvelteKit portfolio for AbdulGafar `</VaryAble>`.

## Stack
- SvelteKit
- Tailwind CSS v4 (`@tailwindcss/vite`)
- TypeScript (strict)
- Vitest + Testing Library
- Playwright
- ESLint (flat config)
- Vercel adapter

## Commands
- `npm run dev`
- `npm run check`
- `npm run lint`
- `npm run test:unit`
- `npm run test:e2e`
- `npm run build`
